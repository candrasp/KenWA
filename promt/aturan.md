selalu ingat aplikasi ini portabel, semua data pada produksi nantinya akan berada di dalam folder yang sama dengan file .exe

contoh kondisi produksi seperti di folder dist_portable.

Kenwa/                    ← Folder distribusi production (tanpa build tools)
├── Kenwa.exe               ← GUI Launcher siap pakai
├── server/                     ← Server Node.js production
├── frontend/                   ← Web dashboard production
└── runtime/                    ← Binary Node.js bundled
