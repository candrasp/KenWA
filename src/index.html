Panduan Penggunaan

1
Mulai Server

Klik tombol Start Server untuk mengaktifkan sistem backend.

2
Pantau Status

Tunggu hingga lampu indikator SRV berwarna hijau dan layar LCD menampilkan trafik data.

3
Akses Dashboard

Setelah status Streaming, klik tombol Buka WhatsApp untuk masuk ke panel utama.

4
Monitoring

Gunakan ikon terminal di pojok kanan atas untuk melihat aktivitas sistem secara real-time.

5
Troubleshooting

Jika server gagal jalan, pastikan tidak ada aplikasi lain yang menggunakan port 3721 atau 3000.

6
Update & Maintenance

Cek pembaruan secara berkala di repository resmi untuk fitur terbaru dan perbaikan bug.

7
Kebutuhan Sistem

Pastikan koneksi internet stabil untuk proses sinkronisasi WhatsApp Web yang lancar.

8
Menutup Aplikasi

Menutup jendela tidak akan mematikan sistem. Aplikasi akan tetap berjalan di System Tray (pojok kanan bawah Taskbar). Klik kanan ikon KenWa dan pilih ❌ Quit KenWa untuk keluar sepenuhnya.
